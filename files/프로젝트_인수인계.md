## ✅ 완성된 것들

### 1. PC용 대시보드 (`us_market_daily_v2.html`)
- 다크 테마, 터미널 스타일 (Oswald + DM Mono + Noto Sans KR 폰트)
- **섹션 구성:**
  - 상단 알림 배너 (Today's Key Driver)
  - 주요 지수 5종 (S&P500, NASDAQ, DOW, VIX, Russell2000) + 스파크라인
  - S&P 500 · NASDAQ 4주 일간 차트
  - 11개 섹터 등락 바차트
  - 시장 심리 지표 6종 (RSI, VIX, CPI, PCE, 실업률, 기준금리)
  - 국채 수익률 곡선 (3M~30Y)
  - 외환 6종 (USD/KRW, DXY, EUR, JPY, GBP, CNY)
  - 원자재 · 크립토 6종 (WTI, 브렌트, 금, 은, BTC, ETH)
  - 버핏(Berkshire) 포트폴리오 TOP 10 + 섹터 도넛 차트 + Q4 변화
  - ARK Innovation ETF (ARKK) TOP 10
  - ARK 오늘의 매매 (Daily Trades)
  - 버핏 vs ARK 비교 바차트
  - Magnificent 7 테이블
  - 주간 TOP 무버 (상승/하락 8종목)
  - 섹터별 관심종목 (AI반도체, 에너지, 금융, 혁신)
  - 시황 해설 5항목 + 한국 투자자 시각

### 2. 모바일용 대시보드 (`market_mobile.html`)
- 완전 모바일 퍼스트 설계 (Syne + JetBrains Mono + Noto Sans KR)
- **UX 구조:**
  - 하단 탭바 6개 (지수 / FX·채권 / 버핏 / ARK / 종목 / 해설)
  - 좌우 스와이프로 페이지 이동 (터치 + 마우스 지원)
  - 페이지 dot 인디케이터
  - iPhone 노치/Safe Area 자동 대응
  - PWA 지원 (홈 화면에 앱처럼 추가 가능)
  - 가로 스크롤 카드 스트립 (지수, Mag7 미니차트)
- **페이지별 내용:**
  - Page 0 (지수): 가로 스크롤 지수 카드 + 4주 차트 + 섹터 차트 + 심리지표
  - Page 1 (FX): 수익률 곡선 + 외환 + 원자재·크립토
  - Page 2 (버핏): 현금 하이라이트 + TOP 10 + Q4 변화 + 섹터 파이
  - Page 3 (ARK): ARKK TOP 10 + 오늘의 매매
  - Page 4 (종목): Mag7 미니차트 + 주간무버 + 워치리스트
  - Page 5 (해설): 시황 해설 4항목 + 내일 이벤트 캘린더

### 3. GitHub Pages 배포 가이드 (`배포가이드.md`)
