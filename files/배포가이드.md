# 📱 미국 시황 대시보드 — 외부 접속 배포 가이드

## GitHub Pages로 무료 배포하기
> 완료하면 `https://[내아이디].github.io/market` 으로 어디서든 접속 가능

---

## ✅ 준비물
- GitHub 계정 (없으면 github.com 에서 무료 가입)
- 대시보드 HTML 파일 (`market_mobile.html`)

---

## 📋 단계별 설정 (5분 소요)

### 1단계 — GitHub 저장소 만들기
1. github.com 접속 후 로그인
2. 우측 상단 **`+`** 버튼 → **New repository** 클릭
3. Repository name: **`market`** 입력
4. **Public** 선택 (중요!)
5. **Create repository** 클릭

---

### 2단계 — HTML 파일 업로드
1. 생성된 저장소 페이지에서 **`uploading an existing file`** 클릭
2. `market_mobile.html` 파일을 드래그 앤 드롭
3. 파일 이름을 **`index.html`** 로 변경
   - 업로드 후 파일명 클릭 → 연필 아이콘 → `index.html` 입력
4. **Commit changes** 클릭

---

### 3단계 — GitHub Pages 활성화
1. 저장소 상단 **Settings** 탭 클릭
2. 왼쪽 메뉴에서 **Pages** 클릭
3. Source → **Deploy from a branch** 선택
4. Branch → **main** / **/ (root)** 선택
5. **Save** 클릭

---

### 4단계 — 접속 확인
- 약 **1~2분 후** 다음 주소로 접속:
  ```
  https://[내GitHub아이디].github.io/market
  ```
- 예시: `https://kimjinho.github.io/market`

---

## 📱 핸드폰 홈 화면에 앱처럼 추가하기

### iPhone (Safari)
1. Safari에서 대시보드 URL 열기
2. 하단 **공유 버튼** (□↑) 탭
3. **홈 화면에 추가** 탭
4. 이름 **"시황"** 입력 → **추가**
5. 홈 화면에 앱 아이콘 생성 ✅

### Android (Chrome)
1. Chrome에서 대시보드 URL 열기
2. 우측 상단 **⋮** 메뉴 탭
3. **홈 화면에 추가** 탭
4. **추가** 확인
5. 홈 화면에 앱 아이콘 생성 ✅

## 💡 자주 묻는 질문

**Q: 비공개로 할 수 있나요?**
A: GitHub Pages는 Public 저장소만 무료입니다. 비공개는 GitHub Pro($4/월) 필요.

**Q: 도메인을 바꿀 수 있나요?**
A: Settings → Pages → Custom domain에서 원하는 도메인 연결 가능.

**Q: 업데이트가 바로 반영 안 돼요**
A: 브라우저 캐시 때문입니다. `Ctrl+Shift+R` (PC) 또는 강력 새로고침으로 해결.
