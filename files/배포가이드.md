# 📱 미국 시황 대시보드 — 외부 접속 배포 가이드 (GitHub Pages & Actions 자동화 적용 버전)

> **알림**: 본 레포지토리는 GitHub Actions를 통한 **자동 시황 데이터 업데이트**가 적용되어 있습니다. 클라이언트의 접속 및 앱 사용은 **GitHub Pages**를 통해 완벽 구동됩니다. 접속 주소 예시: `https://[내아이디].github.io/us-market-board/`

---

## ✅ 1. GitHub Pages 배포 설정 (웹사이트 호스팅 켜기)

GitHub 레포지토리에 푸시된 HTML, CSS, JS 파일들을 온라인에서 볼 수 있는 "웹사이트"로 만들어주는 무료 호스팅 기능입니다.

1. PC 브라우저에서 내 레포지토리로 접속합니다.
   - [👉 레포지토리 바로가기 (wonhyoungpark1983-hash)](https://github.com/wonhyoungpark1983-hash/us-market-board)
2. 상단 탭 맨 오른쪽의 **Settings (톱니바퀴 아이콘)** 탭을 클릭합니다.
3. 왼쪽 메뉴 바에서 **Pages** 메뉴항목을 찾아 클릭합니다.
4. **Build and deployment** 섹션 확인
   - `Source` 부분을 **Deploy from a branch** 로 선택합니다.
   - `Branch` 선택 부분에서 **main** 을 선택하고, 옆 폴더 경로는 **/(root)** 상태 그대로 둔 뒤 **Save** 버튼을 누릅니다.
5. **접속 확인**
   - 약 1~2분 정도 기다린 뒤, 상단에 `Your site is live at https://wonhyoungpark1983-hash.github.io/us-market-board/` 라는 초록색 메시지가 뜨면 완료입니다.
   - 모바일 접속 URL: `https://wonhyoungpark1983-hash.github.io/us-market-board/files/market_mobile.html`
   - PC(데스크톱) 접속 URL: `https://wonhyoungpark1983-hash.github.io/us-market-board/files/us_market_daily_v2.html`

---

## ⚙️ 2. GitHub Actions 자동 데이터 병합 권한 설정 (100% 자동화 달성!)

매일 미국장 마감 직후 한국시간 **오전 6시 30분경(21:30 UTC)**, 봇이 야후 파이낸스에서 데이터를 긁어와 `market_data.json` 파일을 자동으로 **수정 및 커밋**합니다. 이 봇이 동작하기 위해서는 "레포지토리에 수정(Write)" 권한이 필수입니다.

1. **[바로가기 클릭]** 👉 [설정 페이지(Actions General)](https://github.com/wonhyoungpark1983-hash/us-market-board/settings/actions) 로 이동합니다.
2. 마우스 스크롤을 맨 밑으로 내리면 **`Workflow permissions`** 라는 설정 항목이 보입니다.
3. 기본으로 권한이 닫혀있는 `Read repository contents and packages permissions` 대신, 그 위에 있는 **`Read and write permissions`**로 변경하여 체크합니다.
4. 바로 아래에 위치한 **Save** 버튼을 클릭하여 저장합니다.

> 🎉 **위 두 가지 설정을 마치셨다면?**
이제 여러분은 아무 신경을 쓰지 않으셔도 봇이 매일 알아서 데이터를 받아와 코드를 갱신(\`#2\`)하고, GitHub Pages가 그 최신 코드를 사용자들에게 무료로 배포(\`#1\`)하는 **"서버리스 완전 자동화 앱"**의 주인이 되셨습니다!

## 📱 핸드폰 홈 화면에 앱처럼 추가하기

### iPhone (Safari 앱)
1. Safari 브라우저에서 **모바일 접속 URL**을 위로 엽니다.
2. 하단 **공유 버튼** (네모 화살표 모양: □↑)을 누릅니다.
3. 메뉴를 위로 쓸어올려 **홈 화면에 추가**를 탭합니다.
4. 이름에 **"미국 시황"** 등 원하는 이름을 적고 **추가**를 누릅니다.
5. 이제 휴대폰 홈 화면에서 일반 앱처럼 전체화면으로 접속 가능합니다! ✅

### Android (Chrome 브라우저 등)
1. Chrome 브라우저에서 **모바일 접속 URL**을 엽니다.
2. 우측 상단 점 3개 메뉴(**⋮**) 아이콘을 탭합니다.
3. **홈 화면에 추가**를 탭합니다.
4. **추가** 버튼을 눌러 확인합니다.
5. 휴대폰 홈 화면에서 앱 아이콘을 클릭하여 실시간 접속 완료! ✅

## 💡 수동으로 즉시 오늘 시황 업데이트하기

자동 설정된 시간 외에, 지금 당장 최신 상태로 데이터를 긁어오라고 명령하고 싶을 때 사용합니다.

1. [Actions 탭](https://github.com/wonhyoungpark1983-hash/us-market-board/actions)으로 이동.
2. 왼쪽 메뉴에서 **"Daily Market Data Update"** 클릭.
3. 오른쪽 우측 상단쯤의 파란 창 안 **[Run workflow 🔽]** 선택 후 초록색 **[Run workflow]** 클릭.
4. 노란 톱니바퀴가 돌아가다가 몇 초 뒤 초록색 체크마크(✅)가 뜨면 데이터 갱신 완료!
