## 3. Import Order

```typescript
// 1. External libraries
import React from 'react'

// 2. Internal absolute (@/)
import { Button } from '@/components/ui'

// 3. Relative imports
import { useUser } from './hooks'

// 4. Type imports
import type { User } from '@/types'

// 5. Styles
import './styles.css'
```

## 4. Environment Variables

### Naming Convention
| Prefix | Scope | Example |
|--------|-------|---------|
| `NEXT_PUBLIC_` | Client-side | `NEXT_PUBLIC_API_URL` |
| `DB_` | Database | `DB_HOST`, `DB_PASSWORD` |
| `API_` | API keys | `API_SECRET_KEY` |
| `AUTH_` | Authentication | `AUTH_JWT_SECRET` |

### Required Files
- `.env.local` - 로컬 개발용
- `.env.example` - 템플릿 (커밋)
- `lib/env.ts` - 환경변수 검증
